# 📝 To-Do List App  
A simple and interactive To-Do List application built using JavaScript, HTML, and CSS with data persistence using Local Storage.

## 🚀 Features  
- Add new tasks dynamically  
- Mark tasks as completed (toggle)  
- Delete tasks instantly  
- Data stored in Local Storage (persists after refresh)  
- Interactive user interface  

## 🛠️ Tech Stack  
- HTML  
- CSS  
- JavaScript  

## 📚 What I Learned  
- DOM manipulation (createElement, appendChild, classList)  
- Event delegation for dynamic elements  
- Using Local Storage for persistence  
- Handling user input and validation  
- Building dynamic UI  

## 🎯 Future Improvements
